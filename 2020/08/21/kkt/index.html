凸二次规划及其在SVM、SMO算法中的应用

2020-08-21
Word count: 1.3k   |   Reading time≈ 6 min

本文仅探讨形式化为凸二次规划的SVM优化算法，特别地介绍启发式解凸二次规划的SMO算法。

首先介绍SVM（支持向量机）的基本定义。

SVM(支持向量机)

假设输入空间（特征空间）是$\mathcal{X} \subseteq \mathcal{R}^N$，输出空间是$\mathcal{Y}={+1，-1}$。输入$x \in \mathcal{X}$表示实例的特征向量，对应于输入空间（特征空间）的点；输出$y \in \mathcal{Y}$表示实例的类别。给定线性可分/不可分 训练数据集，通过（软）间隔最大化问题或等价地求解相应的凸二次规划问题**学习得到的分离超平面：

$$w^{} \cdot x + b^{} = 0$$

以及相应的分类决策函数
$$f(x) = sign(w \cdot x + b)$$

称为线性支持向量机

sign函数定义

$$
sign(x)=\left{
\begin{aligned}
+1, \qquad x \geq 0 \
-1, \qquad x < 0
\end{aligned}
\right.
$$

凸二次规划

软间隔最大化问题可量化表达为如下凸二次规划问题

原始问题

$$\begin{aligned}
\min {w, b, \xi} & \frac{1}{2}|w|^{2}+C \sum{i=1}^{N} \xi_{i} \
\text { s.t. } & y_{i}\left(w \cdot x_{i}+b\right) \geqslant 1-\xi_{i}, \quad i=1,2, \cdots, N \
& \xi_{i} \geqslant 0, \quad i=1,2, \cdots, N
\end{aligned}$$

我们使用拉格朗日乘子法来简化约束条件

原始问题的拉格朗日函数形式

原始问题约束变形如下

$$\begin{aligned}
\min {w, b, \xi} & \frac{1}{2}|w|^{2}+C \sum{i=1}^{N} \xi_{i} \
\text { s.t. } & -(y_{i}\left(w \cdot x_{i}+b\right) -1 + \xi_i) \leqslant 0, \quad i=1,2, \cdots, N \
& -\xi_{i} \leqslant 0, \quad i=1,2, \cdots, N
\end{aligned}$$

进而得到拉格朗日函数形式

$$L(w, b, \xi, \alpha, \mu) \equiv \frac{1}{2}|w|^{2}+C \sum_{i=1}^{N} \xi_{i}-\sum_{i=1}^{N} \alpha_{i}\left(y_{i}\left(w \cdot x_{i}+b\right)-1+\xi_{i}\right)-\sum_{i=1}^{N} \mu_{i} \xi_{i}$$
其中，$\alpha_i \geqslant 0$，$\mu_i \geqslant 0$，$i=1,\cdots, N $

进而得到优化目标

$$\begin{aligned}
\min {w, b, \xi} \max _{\alpha, \mu}& L(w, b, \xi, \alpha, \mu) \
& \alpha_i \geqslant 0, \quad i=1,2, \cdots, N \
& \mu{i} \geqslant 0, \quad i=1,2, \cdots, N
\end{aligned}$$

该问题又称为拉格朗日极小极大问题

我们记原始问题的最优值为$p^*$

$$p^* = \min _{w, b, \xi} \max _{\alpha, \mu} L(w, b, \xi, \alpha, \mu) $$

对偶问题

拉格朗日对偶问题

$$\begin{aligned}
\max {\alpha, \mu}\min _{w, b, \xi} & L(w, b, \xi, \alpha, \mu) \
& \alpha_i \geqslant 0, \quad i=1,2, \cdots, N \
& \mu{i} \geqslant 0, \quad i=1,2, \cdots, N
\end{aligned}$$

该问题又称为拉格朗日极大极小问题

我们记对偶问题的最优值为$d^$
$$d^ = \max _ {\alpha, \mu}\min _{w, b, \xi} L(w, b, \xi, \alpha, \mu) $$

根据运筹学知识，如下结论成立：

若原始问题和对偶问题都有最优值，则

$$d^{}=\max _ {\alpha, \mu}\min _{w, b, \xi} L(w, b, \xi, \alpha, \mu) \leqslant \min _{w, b, \xi} \max _{\alpha, \mu} L(w, b, \xi, \alpha, \mu)=p^{}$$

推论

设$w^, b^, \xi^, \alpha^, \mu ^$分别是原问题和对偶问题的可行解，并且$d^ = p ^ $，则$w^, b^, \xi^, \alpha^, \mu ^$分别是原始问题和对偶问题的最优解

KKT定理

原始问题

假设 $f(x), c_{i}(x), h_{j}(x)$ 是定义在 $\mathbf{R}^{n}$ 上的连续可微函数。考虑约束最优化问题
$$
\begin{array}{ll}
\min {x \in \mathbf{R}^{n}} & f(x) \
\text { s.t. } & c{i}(x) \leqslant 0, \quad i=1,2, \cdots, k \
& h_{j}(x)=0, \quad j=1,2, \cdots, l
\end{array}
$$
称此约束最优化问题为原始最优化问题或原始问题。

首先，引进广义拉格朗日函数（generalized Lagrange function）
$$
L(x, \alpha, \beta)=f(x)+\sum_{i=1}^{k} \alpha_{i} c_{i}(x)+\sum_{j=1}^{l} \beta_{j} h_{j}(x)
$$
这里， $x=\left(x^{(1)}, x^{(2)}, \cdots, x^{(n)}\right)^{\mathrm{T}} \in \mathbf{R}^{n}, \alpha_{i}, \beta_{j}$ 是拉格朗日乘子， $\alpha_{i} \geqslant 0$ 。考虑 $x$ 的函数:
$$
\min_x \theta_{P}(x)=\min_x \max {\alpha, \beta: \alpha{i} \geqslant 0} L(x, \alpha, \beta)
$$
这里，下标 $P$ 表示原始问题。

对偶问题

可以将广义拉格朗日函数的极大极小问题表示为约束最优化问题:
$$
\begin{array}{c}
\max {\alpha, \beta} \theta{D}(\alpha, \beta)=\max {\alpha, \beta} \min _{x} L(x, \alpha, \beta) \
\text { s.t. } \quad \alpha{i} \geqslant 0, \quad i=1,2, \cdots, k
\end{array}
$$
称为原始问题的对偶问题。

最优解的充要条件

对原始问题和对偶问题，假设$f(x)$和 $c_{i}(x)$ 是凸函数， $h_{j}(x)$ 是仿射函数，并且不等式约束 $c_{i}(x)$ 是严格可行的，则 $x^{}$ 和 $\alpha^{}, \beta^{}$ 分别是原始问题和对偶问题的解的充分必要条件是 $x^{}, \alpha^{}, \beta^{}$ 满足下面Karush-Kuhn-Tucker (KKT) 条件:

$$
\begin{array}{c}
\nabla_{x} L\left(x^{}, \alpha^{}, \beta^{}\right)=0 \
\alpha_{i}^{} c_{i}\left(x^{}\right)=0, \quad i=1,2, \cdots, k \
c_{i}\left(x^{}\right) \leqslant 0, \quad i=1,2, \cdots, k \
\alpha_{i}^{} \geqslant 0, \quad i=1,2, \cdots, k \
h_{j}\left(x^{}\right)=0 \quad j=1,2, \cdots, l
\end{array}
$$
特别指出，$\alpha_{i}^{} c_{i}\left(x^{}\right)=0$称为 KKT 的对偶互补条件。
由此条件可知: 若 $\alpha_{i}^{}>0,$ 则$c_{i}\left(x^{}\right)=0$
